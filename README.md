# trello-kanban-cli

Util that converts Trello Kanban board exports (`Print and Export to JSON`) into clean, structured plain text.

Helps me when describing tasks to LLMs.

It also helps me compare Trello progress against the original feature descriptions written in Obsidian, making it easier to verify which features are still missing or incomplete.
